Replication

Fresh-clone runnable from the repository root

The analysis scripts now resolve paths relative to the checkout instead of a machine-specific absolute directory. The dependency manifest also includes the missing scientific and Excel-reader packages needed by the documented fetch paths. 

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python build_county_lead_master.py
python eagle_lead_suicide_analysis.py

Limits stated plainly

This archive motivates a decisive test; it does not replace it

No individual-level bone lead case-control data

The repository does not contain the individual-level validation study that would directly test the hypothesis in veteran suicide decedents.

NHANES male-only result is null

The README keeps the male-only NHANES null result visible instead of presenting only the favorable pooled estimate.

Community-level findings can still be confounded

The site and README both frame these analyses as ecological and hypothesis-generating, not definitive causal proof.
